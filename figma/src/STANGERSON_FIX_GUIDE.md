# 스탠거슨 대사 문제 해결 가이드

## 🔍 문제 진단

### 발견된 문제들:
1. **내레이션 오인식** - `[홈즈가 스탠거슨을 제압하는...]` 같은 행동 묘사가 캐릭터 대사로 인식됨
2. **캐릭터명 변형** - "조셉 스탠거슨" vs "스탠거슨" 매핑 필요
3. **speaker 속성 불일치** - 노드의 speaker와 실제 대사 화자가 다름

## ✅ 적용된 해결책

### 1. parseDialogue 함수 개선 (완료)
- `[캐릭터명]:대사` 형식 파싱 추가
- 내레이션 자동 판별 로직 추가:
  - `[당신...]` → 내레이션
  - `[홈즈가...]` → 내레이션  
  - `[스탠거슨을...]` → 내레이션
  - `[바닥...]` → 내레이션
  - `[문...]` → 내레이션
  - `[갑자기...]` → 내레이션
  - `[쾅...]` → 내레이션

### 2. 캐릭터 매핑 확장 (완료)
```typescript
const speakerMap = {
  '조셉 스탠거슨': 'stangerson',
  '스탠거슨': 'stangerson',
  '조셉 스탬거슨': 'stangerson',  // 오타 대응
  '스탬거슨': 'stangerson'          // 오타 대응
};
```

## 📝 스탠거슨 노드 파싱 결과 예시

### stangerson_save_count 노드:
```
[당신은 즉시 의사 가방에서 해독제를 꺼냅니다]
→ 내레이션 (narrator)

[왓슨]:아직 늦지 않았습니다!
→ 왓슨 대사 (watson)

[홈즈가 스탠거슨을 제압하는 동안, 당신은 백작을 치료합니다]
→ 내레이션 (narrator)

[모로 백작]:으윽... 왓슨... 박사...
→ 백작 대사 (count)

[셜록 홈즈]:스탠거슨, 당신은 체포됩니다. 백작 살해 미수 혐의로.
→ 홈즈 대사 (holmes)

[조셉 스탠거슨]:...20년이었습니다. 20년간... 나는 두려움 속에 살았습니다...
→ 스탠거슨 대사 (stangerson)
```

## 🎯 테스트 방법

1. **브라우저 콘솔 열기** (F12)
2. **스탠거슨 루트로 진행**:
   - 증거실에서 "유타 장부" 획득
   - 용의자 선택에서 "조셉 스탠거슨 - 공범을 의심한다" 선택
3. **콘솔에서 로그 확인**:
   ```
   [parseDialogue] 노드 stangerson_save_count: 6개 대괄호 대사 발견
   [parseDialogue] [당신은...] (narrator): "즉시 의사 가방에서 해독제를 꺼냅니다"
   [parseDialogue] [왓슨] (watson): "아직 늦지 않았습니다!"
   [parseDialogue] [홈즈가...] (narrator): "스탠거슨을 제압하는..."
   [parseDialogue] [모로 백작] (count): "으윽... 왓슨... 박사..."
   [parseDialogue] [셜록 홈즈] (holmes): "스탠거슨, 당신은 체포됩니다..."
   [parseDialogue] [조셉 스탠거슨] (stangerson): "...20년이었습니다..."
   ```

   const isNarration = speakerLabel.includes('당신') || 
                      speakerLabel.includes('홈즈가') ||
