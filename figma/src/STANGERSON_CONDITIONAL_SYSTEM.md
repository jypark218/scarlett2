# 스탠거슨 발견 전후 조건부 반응 시스템

## 🎯 개요
플레이어가 서재에서 스탠거슨(집사)을 발견했는지 여부에 따라 조사 분위기와 캐릭터 반응이 달라지는 시스템

## 🔑 스탠거슨 발견 확인 조건
```typescript
condition: { visitedNode: 'discover_stangerson' }
// 또는
condition: (context) => context.visitedNodes.includes('discover_stangerson')
// 발견 안 함
condition: (context) => !context.visitedNodes.includes('discover_stangerson')
```

## 📍 적용된 노드들

### 1. 서재 조사 관련 (part2-first-floor.ts, locations/study-room.ts)

#### A. examine_rache (RACHE 글자 조사)
**스탠거슨 발견 전:**
- 홈즈와 왓슨만 조사
- "RACHE... 독일어로 복수를 뜻하지"
- 책장 뒤에서 움직이는 소리 → "...누가 있소?"
- **자유로운 조사 가능**

**스탠거슨 발견 후:**
- 스탠거슨이 불안한 눈빛으로 지켜봄
- 홈즈: "스탠거슨씨, 백작님께서 독일과 관련이 있으신가요?"
- 스탠거슨 창백해짐, 말을 흐림
- **긴장감 있는 조사, 스탠거슨의 반응 관찰 가능**

#### B. safe (금고 조사)
**스탠거슨 발견 전:**
- "금고는 잠겨있다. 4자리 숫자 조합이 필요하다."
- 홈즈: "일기장에 힌트가 있을 수 있네."
- **평온한 조사**

**스탠거슨 발견 후:**
- 홈즈가 금고를 살피자 스탠거슨이 불안한 눈빛으로 다가옴
- 스탠거슨: "그, 그건... 백작님의 금고입니다..."
- 홈즈: "알고 있습니다. 조사가 필요합니다."
- 스탠거슨이 입술을 깨문다. 뭔가 금고 안에 있는 것을 걱정하는 것 같다.
- **스탠거슨이 무언가를 숨기고 있음을 암시**

#### C. examine_desk (책상 조사)
**스탠거슨 발견 전:**
- "오래된 편지들, 장부, 메모들이 흩어져 있다."
- 홈즈: "여기... 비밀번호가 적힌 메모가 있군. '1861'..."
- **조용한 조사**

**스탠거슨 발견 후:**
- 스탠거슨이 긴장한 눈빛으로 지켜봄
- 홈즈가 서랍을 열려고 하자 스탠거슨이 한 걸음 다가옴
- 스탠거슨: "저, 저기... 그건..."
- 홈즈: "조사가 필요합니다, 스탠거슨 씨."
- 비밀번호 "1861" 발견 → 스탠거슨의 얼굴이 창백해짐
- **"1861"이 중요한 의미를 가진다는 암시**

**스탠거슨 발견 후 (지하실 방문 후):**
- 홈즈: "진흙... 지하실의 진흙과 드레버 구두의 진흙. 여관 근처 토양과는 성분이 다를 거야."
- **구체적 증거 연결**

## 🎬 플레이 흐름 예시

### 시나리오 A: 스탠거슨을 먼저 발견
1. 서재 진입 → 책장 뒤 확인 → **스탠거슨 발견!**
2. RACHE 조사 → 스탠거슨이 지켜보며 창백해짐
3. 금고 조사 → 스탠거슨이 불안하게 다가옴 (뭔가 숨기는 듯)
4. 책상 조사 → "1861" 발견 시 스탠거슨 반응 폭발
5. 현관 복귀 → 홈즈: "집사의 태도가 수상해. 더 캐내야 해."
6. **스탠거슨 중심 추리 전개**

### 시나리오 B: 스탠거슨을 나중에 발견
1. 서재 진입 → RACHE/금고/책상 자유롭게 조사
2. 현관 복귀 → 홈즈: "집사를 찾아야 해. 어디 숨어있는 걸까?"
3. 2층/뒷뜰 탐색
4. 다시 서재 → 책장 뒤 확인 → **스탠거슨 발견!**
5. 이미 수집한 증거로 스탠거슨 압박 가능
6. **후반 반전, 퍼즐 완성 느낌**

### 시나리오 C: 스탠거슨을 못 찾음
1. 서재 조사 → 책장 뒤를 확인 안 함
2. 홈즈 힌트: "**서재를 샅샅이 뒤져보게. 숨어있을지도 모르니까.**"
3. 플레이어가 직접 찾아야 함
4. **탐정 게임다운 경험, 성취감**

## 💡 시스템의 효과

### 1. 조사 분위기 변화
- **발견 전**: 조용하고 자유로운 조사, 추리 중심
- **발견 후**: 긴장감, 스탠거슨의 반응 관찰, 심리전

### 2. 힌트 시스템
- 발견 못 하면 홈즈가 직접 힌트 제공
- "서재를 샅샅이 뒤져보게" → 책장 뒤 확인 유도

### 3. 증거의 의미 변화
- **"1861"**: 발견 전엔 그냥 숫자 → 발견 후엔 스탠거슨이 창백해짐
- **금고**: 발견 전엔 퍼즐 → 발견 후엔 "스탠거슨이 뭔가 숨기고 있다"
- **RACHE**: 발견 전엔 추리 대상 → 발견 후엔 스탠거슨의 반응 관찰 가능

### 4. 탐색 순서의 자유도
- 스탠거슨을 먼저 찾을 필요 없음
- 먼저 찾으면 심리전 중심
- 나중에 찾으면 증거 기반 압박 가능
- **플레이어의 선택이 경험을 결정**

## 🎯 스탠거슨 & 지하실 조건 결합

일부 노드는 **두 조건을 동시에 체크**:

### stangerson_timeline_confront (스탠거슨 타임라인 추궁)
```typescript
conditionalText: [
  {
    // 지하실을 이미 방문한 경우 - 더 구체적인 증거로 압박
    condition: (context) => {
      const basementVisited = ['open_basement', 'basement_interior', 'basement_investigation'];
      return basementVisited.some(node => context.visitedNodes.includes(node));
    },
    text: `홈즈: "그리고 저는 지하실을 조사했습니다. 당신 집사복과 똑같은 천 조각이 계단에 걸려있더군요."`
  }
]
```

**경우의 수:**
1. 스탠거슨 발견 X, 지하실 방문 X → 일반 대화
2. 스탠거슨 발견 O, 지하실 방문 X → 스탠거슨 중심 추리
3. 스탠거슨 발견 X, 지하실 방문 O → 지하실 단서 중심
4. 스탠거슨 발견 O, 지하실 방문 O → **양쪽 증거로 강력 압박!**

### 조건부 텍스트 패턴
```typescript
// 패턴 1: 발견 안 한 경우
condition: (context) => !context.visitedNodes.includes('discover_stangerson')

// 패턴 2: 발견한 경우
condition: { visitedNode: 'discover_stangerson' }

// 패턴 3: 지하실 + 스탠거슨 결합
condition: (context) => {
  const basementVisited = ['open_basement', ...];
  const stangersonFound = context.visitedNodes.includes('discover_stangerson');
  return basementVisited.some(node => context.visitedNodes.includes(node)) && stangersonFound;
}
```

| 조건 | 경험 |
|------|------|
| 둘 다 안 함 | 자유로운 조사, 힌트 필요 |

| **둘 다 완료** | **완벽한 압박, 진실 폭로** |
