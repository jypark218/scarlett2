# 🚀 대화 검증기 빠른 시작 가이드

## 🎯 1분 요약

**대화 검증기**는 게임의 모든 대화 노드를 자동으로 검사하여 오류를 찾아주는 도구입니다.

## ⚡ 빠른 사용법

### 방법 1: UI로 검증 (초보자 권장)

```
1. 게임 실행
2. 우측 상단 보라색 톱니바퀴 아이콘 클릭 ⚙️
3. "모든 노드 검증" 버튼 클릭
4. 결과 확인!
```

### 방법 2: 콘솔로 검증 (개발자)

```
1. 게임 실행
2. F12 눌러서 콘솔 열기
3. 자동으로 검증 결과 출력됨!
```

## 🔍 무엇을 검사하나요?

✅ **끊어진 링크** - 존재하지 않는 노드로 연결  
✅ **빈 텍스트** - 대사가 없는 노드  
✅ **잘못된 화자** - 존재하지 않는 캐릭터  
✅ **TODO/TEMP** - 임시 텍스트 감지  

## 🚨 에러 심각도

| 표시 | 이름 | 의미 | 조치 |
|------|------|------|------|
| 🔴 | Fatal | 게임 멈춤 | 즉시 수정 |
| ⚠️ | Error | 버그 발생 | 빠른 수정 |
| ⚡ | Warning | 품질 문제 | 검토 필요 |

## 💡 자주 묻는 질문

**Q: Fatal Error가 뭔가요?**  
A: 게임이 멈출 수 있는 치명적 오류입니다. 즉시 수정하세요!

**Q: Warning은 무시해도 되나요?**  
A: 일부는 의도된 것일 수 있습니다. 하나씩 확인해보세요.

**Q: 검증이 느려요**  
A: 노드가 많으면 몇 초 걸릴 수 있습니다. 정상입니다!
