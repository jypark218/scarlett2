# 대화 톤 통일 가이드

## 캐릭터별 말투

| 캐릭터 | 말투 | 예시 |
|--------|------|------|
| **왓슨 (내레이션)** | 반말체 | "다가간다", "본다", "느껴진다" |
| **스탠거슨 (집사)** | 격식 존댓말 | "~입니다", "~습니다" |
| **호프 (마차꾼)** | 구어체 존댓말 | "~이오", "~하오", "~있소" |
| **드레버 (사업가)** | 구어체 존댓말 | "~요", "~소" |
| **홈즈 (탐정)** | 격식체 | "~입니다" (추리), "~군요" (감탄) |
| **백작** | 격식체 | "~네", "~군" |

## 톤 규칙

### 내레이션
```typescript
✅ "복도 끝에서 담배 연기가 피어오른다."
❌ "복도 끝에서 담배 연기가 피어오릅니다."
```

### 캐릭터 대사
```typescript
// 스탠거슨 (집사 - 격식 유지)
✅ "저는 스탠거슨입니다. 백작님의 집사죠."

// 호프, 드레버 (구어체)
✅ "난 이노크 드레버요."
❌ "저는 이노크 드레버입니다."
```

### 예외
- 홈즈 추리: "~입니다" (전문적 느낌)
- 공식 문서/일기: "~였다", "~했다"
