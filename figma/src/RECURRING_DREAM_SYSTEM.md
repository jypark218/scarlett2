# 🌙 다회차 플레이 시스템 - 반복되는 꿈

## ✅ 구현 완료!

이제 게임을 2회차 이상 플레이할 때, 이전 플레이의 기억이 **꿈처럼** 느껴지는 메타적 연출이 추가되었습니다!

---

## 🎭 작동 방식

### 1회차 플레이
```
인트로 화면 → "게임 시작" → 바로 게임 시작
```
- 일반적인 플레이
- 특별한 연출 없음

---

### 2회차 플레이

```

너무나 생생했다. 마치 실제로 경험한 것처럼...

하지만 그저 꿈일 뿐이겠지.
```

## 🎨 시각적 효과

### 꿈 시퀀스 화면
- 🌙 **보라색/파란색 그라데이션** - 꿈같은 분위기
- ✨ **떠다니는 파티클** (별, 구름, 달)
- 💫 **페이드 인 애니메이션** - 각 대사가 순차적으로 나타남
- ⏱️ **6초 자동 진행** - 읽기 충분한 시간 후 자동으로 게임 시작

### 인트로 화면
- 🌙 **플레이 횟수 표시** (2회차 이상)
  - "🌙 2회차 플레이 중... 이상한 기시감이 든다..."
  - 보라색 테두리, 약간 신비로운 분위기

## 💾 저장 시스템

### localStorage 키
- `moroMansionPlayCount`: 플레이 횟수 저장
- `moroMansionEndings`: 해금된 엔딩 (유지됨)
- `moroMansionCharacters`: 해금된 캐릭터 (유지됨)
- `moroMansionSave`: 현재 플레이 진행상황 (새로하기 시 삭제)

### "새로 하기" 동작
```
✅ 유지되는 것:
- 해금된 엔딩 목록
- 인물 도감
- 플레이 횟수

❌ 삭제되는 것:
- 현재 게임 진행 상황
- 인벤토리
- 방문한 노드
```

## 🎮 사용자 경험 플로우

### 첫 플레이어
```
1. "게임 시작" 클릭
2. 바로 게임 시작
3. 엔딩 도달
4. 인트로로 돌아옴
```

### 2회차 플레이어
```
1. "새로 하기" 클릭
2. 확인 다이얼로그: 
   "현재 진행 중인 게임이 삭제됩니다.
    (해금된 엔딩과 인물 도감은 유지됩니다)"
3. 확인 → 🌙 꿈 시퀀스 재생
4. "어젯밤, 이상한 꿈을 꿨다..."
5. 6초 후 자동으로 게임 시작
6. 인트로에 "🌙 2회차 플레이 중..." 표시
```

### 다회차 플레이어
```
1. "새로 하기" 클릭
2. 확인
3. 🌙 꿈 시퀀스 재생
4. 회차에 따라 다른 대사:
   - 2회차: "이상한 꿈"
   - 3회차: "반복되는 악몽"
   - 4회차: "기시감"
   - 5회차+: "시간의 굴레"
5. 게임 시작
```

## 🎯 메타적 의미

### 스토리와의 연결
이 시스템은 "3명의 용의자" 시스템과 완벽하게 연결됩니다:

> "매번 다른 진실이 펼쳐진다."
> "호프, 스탠거슨, 드레버... 누가 진범인가는 중요하지 않다."

- 플레이어가 여러 루트를 경험할수록
- 꿈 대사가 더 깊은 의미를 가집니다
- 메타적으로 "다회차 플레이"를 인정하는 연출

### 철학적 질문
- 반복되는 시간 속에서 무엇이 진실인가?
- 여러 가능성 중 어느 것이 "진짜" 현실인가?
- 플레이어는 관찰자인가, 참여자인가?
