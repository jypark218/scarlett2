# 지하실 진입 방법 통일 완료

## 🎯 변경 사항
부엌에서 지하실로 가는 방법이 여러 형태로 혼재되어 있던 것을 **"양탄자 밑 숨겨진 문"**으로 통일했습니다.

---

## ❌ 이전 문제점

### 1. kitchen_entrance → basement_entrance
- 텍스트: "방 한구석에 아래로 내려가는 계단이 보인다. 지하실 입구로군."
- 선택지: "지하실로 내려간다"
- **문제**: 계단이 **눈에 보이는** 형태

### 2. basement-route-fix.ts → find_basement
- 텍스트: "부엌 바닥에 숨겨진 지하실 입구를 발견한다."
- **문제**: 어떻게 숨겨져 있는지 불명확

### 3. investigate_kitchen → find_basement
- 텍스트: "홈즈가 바닥 한가운데의 양탄자를 발견한다."
- **정확!**: 양탄자 밑에 숨겨진 문

## ✅ 통일된 형태

### 📍 부엌 입구 (kitchen_entrance)
```typescript
text: `부엌으로 들어선다.

낡은 나무 식탁과 찬장, 그리고 오래된 화덕이 보인다.

홈즈가 주변을 살핀다.

[홈즈]: 부엌치고는 너무 깨끗하군. 마지막으로 사용한 지 며칠이 지난 것 같아.

식탁 위에 빈 접시와 찻잔이 놓여있다.

바닥 한가운데 낡은 양탄자가 깔려있다. 다른 곳은 깨끗한데 그 주변만 진흙이 묻어있다.`

choices: [
  { text: '양탄자를 들춰본다', nextNode: 'basement_entrance' },
  ...
]
```

**포인트:**
- 계단이 눈에 보이지 않음
- 양탄자 주변의 진흙이 힌트
- **플레이어가 의심하고 조사해야 발견**

---

### 📍 양탄자 밑 발견 (basement_entrance)
```typescript
text: `양탄자를 들춰내자... 숨겨진 문이 나타난다!

무거운 철문에 견고한 자물쇠가 달려있다.

[홈즈]: 역시... 지하실 입구가 여기 있었군. 백작이 의도적으로 숨긴 거야.

자물쇠를 살펴본다. 최근에 사용된 흔적이 있다.

[홈즈]: 이 자물쇠... 최근에 열렸던 흔적이 있어. 범인이 이미 안에 있을 수 있어!

문틈 사이로 아래를 들여다보니... 희미한 촛불 빛이 보인다.

[왓슨]: 누군가 아래에 있는 건가?

홈즈가 귀를 기울인다. 하지만 아무 소리도 들리지 않는다.

[홈즈]: 열쇠가 필요해. 백작이 중요한 열쇠는 서재 금고에 보관했을 거야.`
```

**포인트:**
- 양탄자 밑에 **숨겨진 문** 발견
- 최근 사용 흔적 → 범인 암시
- 촛불 빛 → 누군가 아래에 있음
- **금고 열쇠 필요성 힌트**

---

### 📍 NPC 대화 후 진입 (investigate_kitchen)

```typescript
홈즈가 앞장서서 부엌 문을 연다.

[홈즈]: 누군가 의도적으로 흔적을 지웠어. 아마 범인일 거야.

홈즈가 바닥 한가운데의 양탄자를 발견한다.

[홈즈]: 왓슨... 저 양탄자 아래를 봐야 할 것 같아.`

  { text: '양탄자를 들춰본다', nextNode: 'basement_entrance' }

```

**포인트:**
- 스탠거슨 정보로 찾아옴
- 진흙 자국 → 양탄자 의심
- **같은 basement_entrance로 연결**

## 🎭 플레이어 경험

### 발견의 재미
1. **부엌 진입** → "양탄자가 있네?"
2. **진흙 발견** → "주변에만 진흙이... 수상해!"
3. **양탄자 조사** → "숨겨진 문이!"
4. **자물쇠 확인** → "최근에 사용됐어..."
5. **촛불 빛** → "누가 아래에 있다!"

### 긴장감 조성
- 양탄자 = 숨겨진 비밀
- 진흙 = 최근 사용 흔적
- 촛불 = 누군가 있다
- 자물쇠 = 백작의 비밀 공간

---

## 📝 일관성 체크

### ✅ 모든 노드에서 동일
- 지하실 입구 = **양탄자 밑 숨겨진 문**
- 위치 = **부엌 바닥 한가운데**
- 형태 = **무거운 철문 + 자물쇠**
- 힌트 = **진흙, 촛불 빛**

### ✅ 논리적 흐름
1. 백작이 **의도적으로 숨김** (양탄자로 가림)
2. 범인이 **최근 사용함** (자물쇠 흔적)
3. 아래에 **누군가 있음** (촛불 빛)
4. 열쇠는 **금고에 보관** (백작의 중요한 열쇠)

## 🎯 원작과의 연결

### "주홍색 연구"
- 원작에는 지하실 없음
- 모로 백작 저택 = 창작 요소

### "모로 백작의 저택"
- 비밀 통로, 숨겨진 공간
- **양탄자 밑 문 = 전형적인 고딕 호러 요소**
- 백작의 참회 공간 = 지하실
