# 침실 서랍 퍼즐 & 엘렌의 유언장 시스템

## 🎯 퍼즐 플로우

### 1단계: 서재 금고에서 열쇠 획득
```
서재 조사
  ↓
책상에서 비밀번호 획득 (1861)
  ↓
책장에서 금고 발견
  ↓
금고 열기 (safe_puzzle)
  ↓
금고 안에서 3가지 발견:
  - 📒 1861년 장부 (ledger)
  - 🔑 지하실 열쇠
  - 🌹 서랍 열쇠 (drawer_key) ✨ NEW
```

### 2단계: 침실 서랍 조사
```
침실 입장 (bedroom)
  ↓
🔒 서랍장 조사 선택 (bedroom_drawer)
  ↓
자물쇠 발견 (장미 문양)
  ↓
[선택지]
  1. 🔓 서랍 열쇠로 연다 (requiresItem: 'drawer_key')
     → bedroom_drawer_unlocked
  2. 강제로 열어본다
     → bedroom_drawer_force (실패)
```

### 3단계: 유언장 발견
```
서랍 해금 (bedroom_drawer_unlocked)
  ↓
💎 유언장 읽기 (acquire_ellen_will)
  ↓
엘렌 유언장 발견!
  - 백작이 3년 전 작성
  - "엘렌은 내 딸이다"
  - "루시의 딸을 지키겠다"
  ↓
홈즈와 대화 (ellen_will_revelation)
  - 책상의 유언장은 위조
  - 엘렌이 진짜 상속자
  - 루시의 딸이자 백작의 양딸
  ↓
서랍 추가 조사 (bedroom_drawer_more_clues)
  - 엘렌의 편지들
  - 루시의 마지막 편지
  - "제 딸을 부탁합니다..."
```

## 📦 새로 추가된 아이템

### drawer_key (서랍 열쇠)
- **ID**: `drawer_key`
- **이름**: 서랍 열쇠
- **획득 위치**: 서재 금고
- **용도**: 침실 서랍 해금
- **특징**: 장미 문양, 은빛

### ellen_will (엘렌의 유언장)
- **ID**: `ellen_will`
- **이름**: 엘렌에게 남긴 유언장
- **획득 위치**: 침실 서랍 (열쇠로 해금 후)
- **내용**: 
  - 3년 전 작성
  - 전 재산을 엘렌에게
  - "그녀는 내 딸이다"
  - "루시... 미안하오"

## 🔗 스토리 연결

### 기존 유언장 (will)
- 책상 위에 놓여있음
- **위조된 문서**
- 스탠거슨에게 상속

### 진짜 유언장 (ellen_will)
- 서랍에 숨겨져 있음
- **진짜 문서** (3년 전)
- 엘렌에게 상속

### 엘렌 캐릭터
- 루시 페리어의 딸
- 모로 백작의 양딸
- 진짜 상속자
- 현재 저택 어딘가에 숨어있음
