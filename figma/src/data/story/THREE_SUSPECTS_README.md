# 🎭 3명의 용의자 시스템

## 📋 개요

"모로 백작의 비밀"이 대대적으로 업데이트되었습니다!

기존에는 **제퍼슨 호프**만이 진범이었지만, 이제는 **3명의 용의자**가 각각 진범이 될 수 있는 다층적 스토리로 확장되었습니다.

---

## 🔍 3명의 용의자

### 🔴 제퍼슨 호프 (Jefferson Hope)
- **역할**: 마차꾼
- **동기**: 연인 루시 루이자의 죽음에 대한 복수
- **증거**: 📿 루시의 로켓
- **위험도**: ⚠️ 높음
- **진범 여부**: ✅ 진범 (기존 스토리 기반)

**배경**: 1861년 유타에서 모로 백작과 일당이 저지른 금광 사기로 루시의 아버지가 파산했고, 루시는 슬픔 속에 죽었습니다. 20년간 호프는 복수를 계획했습니다.

---

### 🟠 조셉 스탠거슨 (Joseph Stangerson)
- **역할**: 백작의 비서
- **동기**: 과거 공범 관계 은폐, 백작이 자백하려는 것을 막기 위함
- **증거**: 📒 1861년 장부
- **위험도**: ⚠️ 중간
- **진범 여부**: ❓ 플레이어의 선택에 따라 진범이 될 수 있음

**배경**: 1861년 금광 사기의 공범이었던 스탠거슨. 20년간 백작의 비서로 일하며 과거를 숨겨왔습니다. 최근 백작이 양심의 가책으로 경찰에 자백하려 하자, 공황에 빠져 백작을 제거하려 합니다.

---

### 🟡 이녹 드레버 (Enoch Drebber)
- **역할**: 백작의 오랜 친구
- **동기**: 유산 상속, 과거 범죄 은폐
- **증거**: 📜 모로 백작의 유언장 (드레버의 지문)
- **위험도**: ⚠️⚠️ 매우 높음
- **진범 여부**: ❓ 플레이어의 선택에 따라 진범이 될 수 있음

**배경**: 금광 사기의 세 번째 공범. 백작이 재산을 스탠거슨에게 물려주려 하고 과거를 자백하려 한다는 것을 알게 되자, 백작을 죽이고 재산을 차지할 계획을 세웁니다. 가장 위험한 인물.

## 🎮 게임 플레이 흐름

### 1단계: 탐색 (기존 허브 시스템)
```
시작 → 현관 → 서재 / 2층 / 뒷뜰 탐색
```

플레이어는 자유롭게 저택을 탐색하며 단서를 모읍니다.

### 2단계: 증거 수집
플레이어는 **3가지 핵심 증거**를 모아야 합니다:

- 📿 **루시의 로켓** - 우물가에서 획득 (호프의 동기)
- 📒 **1861년 장부** - 서재 금고에서 획득 (스탠거슨의 공범 관계)
- 📜 **모로 백작의 유언장** - 2층 침실에서 획득 (드레버의 유산 탈취 동기)

### 3단계: 최종 추리
3가지 증거를 모두 모으면 **최종 추리 허브**로 진입합니다.

```
final_deduction_hub → suspect_choice_hub
```

홈즈와 왓슨이 모든 단서를 정리하고, 플레이어가 누구를 주요 용의자로 볼지 선택합니다.

### 4단계: 용의자 루트 선택
```
🔴 호프 루트 → 지하실 대면 → 설득 / 대립
🟠 스탠거슨 루트 → 비서실 급습 → 백작 구출
🟡 드레버 루트 → 드레버의 함정 → 전투
```

각 루트마다 다른 긴장감과 선택지가 제공됩니다.

## 🎯 10가지 엔딩

### 🌟 진엔딩 (1개)
**#1: 자비와 구원** (`true_ending_mercy`)
- 조건: 호프 루트 선택 + 설득 성공 + 호프가 백작을 용서
- 결과: 모든 사람이 구원받고, 피해자가 더 이상 발생하지 않음
- 루시의 영혼도 평화를 얻음

---

### ✅ 굿 엔딩 (2개)

**#2: 복수자의 눈물** (`good_ending_hope`)
- 조건: 호프 루트 + 설득 성공 (용서는 하지 않음)
- 결과: 백작은 살고, 호프는 법적 처벌, 진실이 밝혀짐

**#3: 공범의 고백** (`good_ending_stangerson`)
- 조건: 스탠거슨 루트 + 백작 구출 성공
- 결과: 스탠거슨 체포, 백작 생존, 20년 비밀 공개

---

### ❌ 배드 엔딩 (7개)

#### 호프 관련 (3개)

**#4: 탐정의 실수** (`bad_hope_wrong_deduction`)
- 조건: 호프가 진범이 아니라고 잘못 판단
- 결과: 백작 독살당함

**#5: 무모한 돌진** (`bad_hope_violence`)
- 조건: 호프와 물리적 충돌 선택
- 결과: 왓슨 사망

**#6: 가혹한 비난** (`bad_hope_count_suicide`)
- 조건: 백작을 과도하게 비난
- 결과: 백작 자살

#### 스탠거슨 관련 (2개)

**#7: 배신자의 승리** (`bad_stangerson_betrayal`)
- 조건: 스탠거슨을 의심하지 않음
- 결과: 백작 사망, 스탠거슨 도주, 호프 누명

**#8: 무고한 희생** (`bad_stangerson_innocent_dies`)
- 조건: 잘못된 추리로 드레버를 쫓는 사이...
- 결과: 스탠거슨이 백작을 살해

#### 드레버 관련 (2개)

**#9: 숨겨진 진실** (`bad_drebber_hidden_truth`)
- 조건: 드레버가 진범임을 모름
- 결과: 드레버가 백작 살해 + 호프에게 누명 + 악의 승리

**#10: 친구의 죽음** (`bad_holmes_dies`)
- 조건: 드레버의 함정에 걸림
- 결과: 셜록 홈즈 사망

## 🎨 디자인 철학

### 다층적 미스터리
- **1차원**: "누가 범인인가?" (기존)
- **2차원**: "어떻게 막을 것인가?" (선택)
- **3차원**: "진정한 정의는 무엇인가?" (철학)

### 의미 있는 선택
모든 선택이 결과에 영향을 미칩니다:
- 증거 수집 여부
- 용의자 선택
- 대화/설득/폭력 선택
- 도덕적 판단

### 감정적 깊이
- 호프: 슬픔과 복수
- 스탠거슨: 두려움과 배신
- 드레버: 탐욕과 잔인함
- 왓슨: 인간미와 의사의 사명
