# 🕯️ 참회실 루트 - 우물 전용 경로

## 🎯 개요
우물에서만 접근 가능한 백작의 비밀 참회실 루트를 추가하여, 부엌과 우물 두 입구의 차별화를 완성했습니다.

---

## 🗺️ 지하실 구조

```
[부엌 양탄자]          [우물 로프]
     ↓                    ↓
  [철문]              [철문 (우물 벽)]
     ↓                    ↓
     │               [참회실]
     │                ├─ 백작 일기
     │                ├─ 루시 사진
     │                ├─ 로켓 발견
     │                └─ 참회 편지
     │                    ↓
     │               [통로 발견]
     │                    ↓
     └─────→ [지하실 메인 홀] ←┘
                    ↓
              [백작 감금실]
                    ↓
            [basement_scene]
```

---

## 🎮 두 루트 비교

### 부엌 루트 (메인 루트)
**특징:** 빠르고 직선적, 액션 중심

```
부엌 → 양탄자 → 철문 → 지하실 메인 홀 → basement_scene
```

**경험:**
- 긴박함: "범인이 아래에 있어!"
- 직접적: 바로 메인 이벤트
- 액션: 호프와 대결, 백작 구출

**텍스트:**
- "무거운 철문에 자물쇠가 걸려있다"
- "최근에 열렸던 흔적이 있어"
- "촛불 빛이 보인다"

---

### 우물 루트 (서브 루트)
**특징:** 깊이 있고 감성적, 진실 중심

```
우물 → 로프 내려가기 → 철문 → 참회실 → 통로 → 지하실 메인 홀 → basement_scene
```

**경험:**
- 탐험: "우물 안에 문이?"
- 발견: 백작의 사적 공간
- 감성: 루시, 죄책감, 참회
- 보상: 로켓을 여기서 먼저 획득 가능

**참회실 콘텐츠:**
1. **백작의 일기** - 20년간의 죄책감
2. **루시의 사진** - 호프와 함께 찍은 행복한 시절
3. **루시의 로켓** - 바닥에 떨어진 은빛 로켓 (아이템 획득)
4. **참회 편지들** - 백작이 스스로를 벌한 흔적
5. **십자가** - "Vergebung ist unmöglich" (용서는 불가능하다)

## 📝 노드 흐름

### 1. 우물에서 발견
**well_deep** (수정됨)
```
"우물 아래를 내려다보니... 희미하게 문이 보인다."
→ "로프를 만들어 내려간다"
```

### 2. 로프 하강
**well_descend**
```
"홈즈가 우물 벽을 살펴보더니 말한다.
벽에 홈이 파여있어. 발판으로 쓸 수 있겠어."
```

**well_rope_descent**
```
"조심스럽게 밧줄을 잡고 우물 안으로 내려간다.
5미터쯤 내려가자... 우물 벽 옆에 작은 철문이 나타난다."
```

### 3. 참회실 진입
**confession_room_entrance**
```
"석조 벽, 낮은 천장, 그리고 중앙에 놓인 낡은 나무 책상.
촛불이 타고 있다. 누군가 최근에 이곳을 사용했다.
[홈즈]: ...참회실이군."
```

**선택지:**
- 책상 위 일기장을 조사한다
- 벽의 사진들을 살펴본다
- 편지들을 읽어본다
- 십자가를 조사한다

### 4. 참회 일기
**confession_diary**
```
1871년 1월 15일
"주여, 저를 용서하소서. 루시를 잃었습니다. 제 탐욕이 그녀를 죽였습니다."

1891년 12월 20일 (며칠 전)
"제퍼슨 호프가 왔습니다. 복수의 시간입니다. 저는... 도망치지 않겠습니다."
```

### 5. 루시의 사진들
**confession_photos**
```
첫 번째 사진: 젊은 여성의 초상화. "Lucy Louisa Ferrier - 1860"
두 번째 사진: 루시와 제퍼슨 호프가 함께. 행복해 보인다.
세 번째 사진: 모로 백작, 드레버, 스탠거슨. 유타 1861년.

바닥에 떨어진 로켓을 발견한다...
```

### 6. 로켓 획득
**confession_find_locket**
```
[아이템 획득: 루시의 로켓]

펼쳐보니 안에 작은 초상화가 들어있다. 루시의 얼굴...
뒷면: "Forever - J.H."

[홈즈]: 호프가 루시에게 준 로켓이야. 백작이 어떻게 이걸 갖고 있었을까?
```

### 7. 참회 편지들
**confession_letters**
```
루시가 호프에게 쓴 편지 (미완성):
"백작이 아버지를 속였어요... 치료비가... 제발 와주세요..."

백작이 자신에게 쓴 편지:
"모로, 너는 악마다. 루시를 죽인 건 네 탐욕이다."
```

### 8. 십자가
**confession_cross**
```
"Vergebung ist unmöglich" (용서는 불가능하다)

십자가 밑에 호프의 협박장들:
"백작, 기억하시오. 루시가 당신 때문에 죽었소."
"복수의 날이 올 것이오."
```

### 9. 통로 발견
**confession_hallway_discovery**
```
반대편 벽에 작은 문이 있다.
[홈즈]: 왓슨, 통로가 있어.
이 통로는... 아마 메인 지하실로 연결될 거야.
```

### 10. 메인 홀 도착
**basement_hallway**
```
넓은 지하 홀에 들어선다.
양옆으로 여러 방들이 보인다.
왼쪽에 작은 방이 보인다. 문이 반쯤 열려있다.
안에서... 희미한 신음 소리가 들린다.
```

### 11. 메인 이벤트
**basement_scene**
```
(부엌 루트와 합류)
지하실에 들어서자 놀라운 광경이 펼쳐진다.
모로 백작이 결박되어 있고, 호프가 약병을 놓고 있다...
```

## 🎭 플레이어 경험 비교

### 시나리오 A: 부엌 먼저
```
1. 부엌 → 양탄자 발견 → 지하실 입구
2. 열쇠 필요 → 금고 찾기
3. 열쇠 획득 → 지하실 진입
4. basement_scene → 호프와 대결
5. 백작 구출 → 진실 밝히기
```
**느낌:** 빠른 액션, 긴박한 전개

---

### 시나리오 B: 우물 먼저
```
1. 우물 → 편지 발견 → 로프 하강
2. 참회실 → 백작의 과거 탐험
3. 일기, 사진, 로켓 발견
4. "백작은 평생 괴로워했구나..."
5. 통로 → 메인 홀 → basement_scene
6. "아, 그래서 백작이 도망가지 않았구나!"
```
**느낌:** 깊이 있는 스토리, 감정 이입, 백작에 대한 이해

---

### 시나리오 C: 양쪽 다 탐험
```
1. 우물 → 참회실 → 백작의 참회 이해
2. "통로가 있네? 일단 돌아가자"
3. 부엌 → 양탄자 → "아, 이 통로 저기로!"
4. basement_scene → 모든 정보를 갖춘 상태
5. **완벽한 이해와 선택**
```
**느낌:** 완벽한 탐정, 모든 퍼즐 완성

## 🎊 결과

### 이전 문제
- 부엌과 우물이 같은 곳으로 → 두 번째 발견 시 "또 여기네..."
- 우물이 의미 없음 → "왜 우물로 가야 해?"

### 해결
- **부엌 = 빠른 액션 루트**
- **우물 = 깊이 있는 스토리 루트**
- 양쪽 다 가치 있음
- 플레이어 선택에 따라 경험 달라짐

### 플레이어 반응 예상
1. "우물에서 내려가니 비밀 공간이!"
2. "백작의 일기를 읽으니 불쌍하네..."
3. "로켓을 여기서 먼저 찾았어!"
4. "아, 통로로 메인 지하실과 연결되는구나!"
5. "부엌으로 갔으면 이걸 못 봤겠네!"

## 🎯 최종 평가

### 차별화 성공 ✅
- 우물 루트 = 참회실 (과거, 감성)
- 부엌 루트 = 메인 홀 (현재, 액션)

### 지루함 해소 ✅
- 같은 곳 아님
- 다른 경험 제공
- 양쪽 다 탐험할 가치

### 퍼즐 유지 ✅
- 우물 퍼즐 그대로
- 로프 내려가기 추가
- 탐험 재미 증가

### 스토리 깊이 ✅
- 백작 캐릭터 깊이 증가
- 루시-호프 관계 구체화
- 참회와 복수 주제 강화
