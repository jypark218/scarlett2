- [ ] 선택지 3개만 표시됨:

  - [ ] "🖼️ 벽면 전체를 관찰한다"
  - [ ] "👣 바닥과 주변을 관찰한다"

- [ ] **책상 관찰** 클릭 → 투쟁 흔적 발견

- [ ] "🔍 정밀 조사를 시작한다" 버튼 활성화 확인

### Phase 2: 정밀 조사
- [ ] study_phase2_hub 진입
- [ ] 조건부 선택지 표시 확인:
  - [ ] "📜 RACHE 핏자국을 정밀 조사한다" (벽면 관찰 후에만)
  - [ ] "🗂️ 가계도를 정밀 조사한다" (벽면 관찰 후에만)
  - [ ] "📖 책상 서랍을 정밀 조사한다" (책상 관찰 후에만)

## 🐛 오류 확인

### 노드 미발견 오류
```
❌ Error: Node 'xxx' not found
→ /data/story/locations/index.ts 확인
→ Export 누락 확인
```

### 선택지 미표시
```
❌ 선택지가 안 보임
→ requiredVisitedNode 조건 확인
→ 플래그 설정 확인
```

### 무한 루프
```
❌ 같은 노드 반복
→ nextNode 연결 확인
→ hideIfVisitedNode 추가 확인
```
