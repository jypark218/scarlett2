## 🎭 구현된 진엔딩: "진실의 무게"

### 스토리 개요
```
루시의 편지를 통해 호프가 복수를 멈춘다.
엘렌, 호프, 백작 세 사람이 화해한다.
백작은 자수하지만 엘렌과 호프가 용서한다.
진정한 구원은 복수가 아닌 용서에서 온다.
```

### 핵심 대사
```
[제퍼슨 호프]: "...루시. 당신을 닮은 이 아이의 눈에
               어둠을 심어줄 수는 없겠군."

[엘렌]: "아버지... 백작님이 저를 사랑했다는 걸... 알아요.
        20년간... 속죄하셨다는 것도..."

[왓슨의 수기]: "복수가 아닌 용서를. 증오가 아닌 화해를.
               이것이 진정한 구원이었다."
```

### 도달 조건
```
1. ✅ 엘렌과 만남 (ellen_appears)
2. ✅ 루시의 편지 획득 (lucy_letter)
3. ✅ 지하실 진입 (find_basement)
4. ✅ 루시 편지 제시 (show_lucy_letter_basement)
5. ✅ 호프의 자비 선택 (hope_mercy_route)
6. ✅ 화해 성공 (reconcile_all_three)
```
